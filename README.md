# Flashzilla
Flashzilla es una app desarrollada en SwiftUI que simula un juego de repaso utilizando tarjetas. Permite a los usuarios agregar sus propias preguntas y respuestas, repasar las tarjetas, marcar respuestas correctas o incorrectas, y reiniciar el juego.
